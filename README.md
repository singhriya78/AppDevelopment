#  Responsive Sidebar Menu

A clean, responsive sidebar navigation menu built with pure HTML and CSS — no JavaScript required for the toggle functionality (uses the checkbox hack).

## Features

- Pure CSS toggle (checkbox-based, no JS dependency)
- Responsive hamburger menu with open/close icons
- Smooth slide-in/slide-out animation
- Font Awesome icons for menu items and social links
- Google Fonts integration (Gentium Plus, Oswald, Poppins)
- Clean, minimal UI ideal for portfolios, college/club websites, or dashboards

## Built With

- HTML5
- CSS3
- [Font Awesome](https://fontawesome.com/) — icons
- [Google Fonts](https://fonts.google.com/) — typography

## 📋 Menu Sections

- Gallery
- Shortcuts
- Exhibits
- Events
- Store
- Contact
- Feedback
- Social Media links (Facebook, Twitter, Instagram, YouTube)

## ⚙️ How It Works

The sidebar uses a hidden `<input type="checkbox">` paired with `<label>` elements to toggle visibility — a common "CSS checkbox hack" that avoids JavaScript for simple show/hide interactions.
